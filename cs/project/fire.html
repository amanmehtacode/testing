UX designer
UI/UX & Brand Design

The User Interface

The UI is a dark themed design, followed up with the branded orange colour and gradient for highlights. It's a minimal design, led by whitespace.

My biggest challenge was trying to compact a whole load of features in a minimal UI, without it looking complicated. I was able to do this by creating a design system that prioritizes big features and content using Visual Hierarchy. It worked out great and came all together to make that amazing & fresh UI.

Visual Hierarchy
